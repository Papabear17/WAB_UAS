## üöÄ Installation & Usage

### Prerequisites
-   [Node.js](https://nodejs.org/) installed on your machine.

### Running Locally

1.  **Clone or Download** the repository.
2.  **Navigate** to the project directory in your terminal.
3.  **Start the server**:
    ```bash
    node server.js
    ```
4.  **Open your browser** and visit:
    -   http://localhost:3000

Alternatively, you can simply open the `index.html` file directly in your browser, though some features might work better on a server.
